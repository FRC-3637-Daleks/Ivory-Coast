# Ivory Coast

### Explaining the Naming convention
This name was chosen to create a new naming scheme for outreach bot and other non-main bots. Use that year's alphabetical letter (in this case, 2025 is I), and make it correspond with the letter of a country (in this case Ivory Coast)
